# Fin-clamp
This repository provides a list of materials, equipment and instructions to produce the fin-clamp described in the article “A New Reproducible, Minimally Invasive Fin-Clamp Enabling Biologging and Animal-Borne Video Studies of Juvenile Sharks”.
